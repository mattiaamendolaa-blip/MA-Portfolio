# Product

## Register

MA.dev

## Users

Recruiter, hiring manager e potenziali clienti che valutano competenze tecniche e senso estetico di un full-stack developer. Arrivano da LinkedIn, GitHub o ricerca diretta. Navigano velocemente, spesso da mobile, cercando prove di competenza e un modo rapido per contattare.

## Product Purpose

Portfolio personale di Mattia Amendola. Mostra progetti, competenze tecniche e certificazioni. Il design stesso è prova di competenza frontend. Successo = il visitatore capisce chi è Mattia, cosa sa fare, e lo contatta.

## Brand Personality

Tecnico, elegante, professionale. Competenza ingegneristica con cura estetica. Pulito senza essere sterile, moderno senza essere caotico. Ogni dettaglio è intenzionale.

## Anti-references

- Template generici (WordPress starter themes, Bootstrap portfolio templates, Vercel templates non customizzati)
- Portfolio "tutti uguali" con hero centrato + 3 card feature identiche + footer minimalista
- Overdesign: troppe animazioni, gradienti AI-purple, neon glow ovunque
- UI kit non customizzati (shadcn/ui default state, Tailwind UI copia-incolla)

## Design Principles

1. **Mostra, non dichiarare.** Il codice interattivo (3D 404, dock Skills, NFC tap) dimostra competenza meglio di qualsiasi lista di skill.
2. **Precisione ingegneristica.** Ogni pixel, ogni transizione, ogni stato hover è deliberato. Nessun dettaglio lasciato al caso.
3. **Chiarezza immediata.** Il visitatore capisce cosa fa Mattia in 5 secondi. Nessuna ambiguità, nessun gergo inutile.
4. **Coerenza cross-theme.** Light e dark mode sono entrambi di prima classe, non l'uno sottoprodotto dell'altro.
5. **Performance come feature.** Animazioni a 60fps, zero layout shift, caricamento rapido. La velocità è parte dell'esperienza.

## Accessibility & Inclusion

- WCAG AA come minimo, AAA target per testo corpo
- `prefers-reduced-motion` rispettato con media query globale
- `prefers-reduced-transparency` fallback sui glassmorphism
- Focus states visibili su tutti gli interattivi
- Skip-to-content link presente
- `lang` attributes corretti su testi misti italiano/inglese
